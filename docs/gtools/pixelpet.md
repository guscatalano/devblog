## What it does

- **Transparent, always-on-top pet** with per-pixel interaction — only the cat's
  pixels catch the mouse; empty space stays click-through.
- **Lives on its own** — a personality-weighted ambient loop makes it wander, nap,
  and idle.
- **Drag it anywhere**, click for a reaction, hover to greet it. Settings live in
  the system tray.
- **AI-generated pets** from your own photos are on the roadmap (optional, using
  the AI provider of your choice).

Early development — the default white cat works today. Not yet code-signed, so
SmartScreen may warn on first run (**More info → Run anyway**).

See the [privacy notes](/privacy/pixelpet) for the optional AI feature and how
API keys are stored.
