# Privacy Policy — RDP DVC Watcher

_Last updated: July 12, 2026 · [All privacy policies](/privacy/) · [Source](https://github.com/guscatalano/RDP_DVC_Watcher)_

**Summary: RDP DVC Watcher is a local diagnostics tool. It prints RDP channel
traffic statistics to the console and stores nothing, sends nothing, and
contains no telemetry.**

## What the app works with

RDP DVC Watcher is a console tool that opens a local ETW trace session for the
Remote Desktop server provider and tallies per–dynamic-virtual-channel byte
counts. It reads only channel names and traffic sizes — no message content, no
personal data. It requires Administrator rights to run ETW.

## What leaves your device

**Nothing.** No network connections, no telemetry, no analytics, no update
check.

## What's stored on your device

**Nothing.** Counts are held in memory and printed to the console. No files,
logs, caches, or registry writes.
